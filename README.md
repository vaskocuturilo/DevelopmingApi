## Development API

You will need the following technologies available to try it out:
* Git
* Maven 3+
* JDK 8
* IDE of your choice 
* Web browser Chrome

### How to run

```gradle clean test```

### Generate Allure report 

```gradle allureReport```

### Open Allure report in browser

```gradle allureServe```
